## 一、产品概述

### 一句话定位
随时召唤世界顶级思想家，以他们的心智模型为你答疑解惑。

### 解决的问题
用户在面对创业、人生、写作、投资等决策时，缺乏具有深度和独特视角的思考伙伴。泛化的 AI 回答无法提供带有特定认知风格的回应。

### 目标用户
- 创业者、产品经理：需要来自顶级思想家视角的建议
- 写作者、思考者：希望获得有思想深度的对话伙伴
- 终身学习者：希望通过对话内化顶级思想家的思维框架

## 二、MVP 功能范围

### 已实现功能

**F1 — 智囊团主页**
- 展示 4 位思想家卡片（头像、姓名、职位标签、领域标签、一句话描述）
- 卡片 hover 效果 + 点击跳转对话页
- 中英文切换按钮（右上角）
- Footer 展示创作者社交链接（LinkedIn、GitHub、Email、微信复制）

**F2 — 人物对话页**
- 对话界面展示人物身份标识（头像、姓名、职位）
- 系统自动加载对应人物的 SKILL.md 作为 system prompt
- 流式输出（streaming），逐字显示回复
- Markdown 渲染（支持表格、列表、加粗、引用等）
- 支持新建对话（清空历史）
- 支持返回主页
- 空状态页展示 3 条推荐问题

**F3 — 中英文切换**
- 全局语言状态，`localStorage` 持久化
- 切换后 UI 文案（标题、标签、按钮、推荐问题）全部响应
- AI 回复语言跟随切换
- 切换不清空对话历史

**F4 — 错误处理**
- API 调用失败展示友好提示
- 支持重试
- 头像加载失败显示字母 + 色块 fallback

### MVP 不包含
- 用户登录 / 账号系统
- 对话历史持久化（刷新后消失）
- 多人物同时对话（群聊模式）
- 付费墙 / 用量限制
- 移动端 App / 微信小程序

## 四、技术架构

### 技术选型

| 层 | 技术 | 选择原因 |
|----|------|---------|
| 前端框架 | Next.js 16 (App Router) | SEO 友好，Vercel 部署无缝 |
| 样式 | Tailwind CSS v4 | 快速开发，无需维护 CSS 文件 |
| AI 调用 | Vercel AI SDK v3 + DeepSeek | 流式输出支持好，DeepSeek 成本低 |
| 部署 | Vercel | 自动 CI/CD，免费额度够用 |
| 人物技能存储 | 本地 .md 文件 | 简单直接，修改即生效 |

### 数据流

```
用户选择人物 → 加载 SKILL.md → 作为 system prompt
→ 用户输入 + language 参数 → POST /api/chat
→ DeepSeek API (streaming) → 流式返回 → 逐字渲染
```

### 页面路由

```
/                    → 主页（人物选择）
/chat/[characterId]  → 与特定人物的对话页
/api/chat            → POST，流式对话 API
```

## 六、扩展性设计

### 新增人物
1. 在 `src/skills/` 放入 `xxx.md`
2. 在 `src/config/characters.ts` 新增一条配置

无需修改路由、API、或任何其他文件。

### 新增语言
1. 在 `src/i18n/translations.ts` 新增语言对象
2. 更新 `LanguageContext` 的类型
